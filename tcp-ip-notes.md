# What is TCP/IP?

TCP/IP is the set of rules that lets computers talk to each other over the internet.

- **IP (Internet Protocol):** The "postal address" that tells data where to go.
- **TCP (Transmission Control Protocol):** The "postal worker" that checks if the data arrived safely.

## Key Takeaway
If IP is the address on the envelope, TCP is the tracking number that ensures the package isn't lost.
